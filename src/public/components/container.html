Container

A wrapper component for restricing the width of your main content.

Default container

The default container spans the full width of the viewport with padding on both sides.

Sizes

There are four size options ranging from widest to narrowest. Choose whichever size suits best for each situation. Multiple different sizes can be used per screen. One factor which helps decide the most appropriate size is the line length of the content it contains.

Responsive sizes

A container can adapt to the width of the viewport by specifying responsive size classes prefixed with bp1-, bp2-, bp3- or bp4-. Resize your browser window to see the Container below adapt.
