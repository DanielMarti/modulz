Visibility

Control the visibility of elements without modifying their display value. Useful for temporarily hiding content while still keeping it available to screen readers.

Class Description
.visibility-visible

Set an element box to be visible.
.visibility-hidden

Set an element to be invisible. Layout will not be affected. The element cannot receive focus (such as when navigating through tab indexes).
