Spacing

A set of utilities for adding space to layouts.

Margin

Add space around elements.

Class Description
.margin-0

Set the margin to 0 on all sides of an element.
.margin-3

Set the margin to a specific value on all sides of an element. Use any value from 0 through 9.
.mt-3

Add a margin to a specific side using mt-3, mr-3, mb-3 or ml-3. Use any value from 0 through 9.
.ml-auto

Set the margin to auto on a specific side of an element.
.bp2-mr-3

Responsive classes for setting margins only at specific breakpoints. There are 4 breakpoint prefixes bp1-, bp2-, bp3- and bp4-.

Padding

Add space inside elements.

Class Description
.padding-0

Set the padding to 0 on all sides of an element.
.padding-3

Set the padding to a specific value on all sides of an element. Use any value from 0 through 9.
.pt-3

Add a padding to a specific side using pt-3, pr-3, pb-3 or pl-3. Use any value from 0 through 9.
.bp2-pr-3

Responsive classes for setting padding only at specific breakpoints. There are 4 breakpoint prefixes bp1-, bp2-, bp3- and bp4-.

Negative margin

Add negative space around elements to pull elements in a specific direction.

Class Description
.nmt-3

Add a negative margin to a specific side using nmt-3, nmr-3, nmb-3 or nml-3. Use any value from 1 through 9.
.bp2-nmr-3

Responsive classes for setting negative margins only at specific breakpoints. There are 4 breakpoint prefixes bp1-, bp2-, bp3- and bp4-.
