Typography

Text utilities for controlling alignment, weight, leading, style etc.

Alignment

Utilities for controlling text alignment.

Class Description
.ta-left

Align text to the left
.ta-center

Align text to the center
.ta-right

Align text to the right

Leading

Utilities for controlling line-height.

Class Description
.lh-1

Apply the 1st value in the leading scale
.lh-2

Apply the 2nd value in the leading scale
.lh-3

Apply the 3rd value in the leading scale
.lh-4

Apply the 4th value in the leading scale
.lh-5

Apply the 5th value in the leading scale
.lh-6

Apply the 6th value in the leading scale
.lh-7

Apply the 7th value in the leading scale
.lh-8

Apply the 8th value in the leading scale
.lh-9

Apply the 9th value in the leading scale

Font style

Utilities for controlling font style.

Class Description
.fs-normal

Set a font in normal characters
.fs-italic

Set a font in its italic characters. If italic characters are unavailable, the style is artificially simulated.
