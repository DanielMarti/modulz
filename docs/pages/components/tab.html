Tab

Tabs allow the user to toggle different sets of content within the same container.

Tab structure

Use a Flexbox component to place tabs in a single row above their associated content. Each tab's label should succinctly describe its corresponding content. You can apply an active state using the is-active modifier class. Exactly one tab must be active at any time.
