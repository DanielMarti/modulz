Divider

A small separator which represents a presentational break between element groups—e.g. separate a log in link from a group of navigation links.

Default divider

Define a Divider using an empty div. Don't use a hr because it's inappropriate for many use cases.

Sizes

There are 3 sizes available plus an expanded variation, which spans the full width of its container. The default size is size2.

Colors

Colors are handled with background-color utility classes. Check out the color docs for more detailed guidelines on color usage.

Vertical divider

Each of the sizes is also available in a vertical orientation by adding the vertical modifier class.
